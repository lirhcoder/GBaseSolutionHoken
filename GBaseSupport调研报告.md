## 1. 产品概述

### 1.1 产品定位
GBase Support 是由日本 Sparticle 株式会社开发的**AI知识代理（AIナレッジエージェント）**产品，专注于为企业提供基于自有数据构建响应模型的AI聊天机器人解决方案。

### 1.2 产品背景
- **开发公司**：Sparticle株式会社（东京都中央区日本桥小传马町，成立于2019年7月）
- **正式发布**：2025年6月11日
- **最新版本**：v3.4.5（2025年12月19日）
- **产品系列**：GBase Support 是 GBase 产品系列中面向客户支持场景的解决方案

### 1.3 技术基础
- 基于生成式AI和RAG（检索增强生成）技术融合
- 采用大规模语言模型（LLM）
- 支持 bge m3 嵌入模型
- 集成 qdrant 向量数据库，支持动态索引参数

## 2. 核心功能分析

### 2.1 AI聊天机器人功能

| 功能模块 | 功能描述 |
|---------|---------|
| **ノーコード构建** | 无需编程即可创建和管理AI聊天机器人 |
| **知识库学习** | 上传文档后AI自动学习，支持URL、PDF、Word、PPT等多种格式 |
| **信息源明示** | AI生成的回答会明确显示信息来源 |
| **24小时自动应答** | 全天候自动化客户支持服务 |

### 2.2 多语言支持（10种语言）

- **自动翻译功能**：日语FAQ自动翻译为多国语言
- **支持语言（10种）**：
  1. 日本語（日语）
  2. English（英语）
  3. 中文（中国语）
  4. 한국어（韩语）
  5. Tiếng Việt（越南语）
  6. Português（葡萄牙语）
  7. Português (Brasil)（巴西葡萄牙语）
  8. Bahasa Indonesia（印度尼西亚语）
  9. नेपाली（尼泊尔语）
  10. Español（西班牙语）
- **版本历史**：
  - v3.3.4 (2025/08/21): 新增葡萄牙语、巴西葡萄牙语、尼泊尔语、印度尼西亚语、西班牙语
  - v3.3.5 (2025/08/29): 新增越南语
- **未来扩展**：可根据需求继续增加泰语、菲律宾语、阿拉伯语等

### 2.3 渠道集成

| 集成渠道 | 说明 |
|---------|------|
| **网站嵌入** | 可嵌入企业官网或内部门户 |
| **LINE Bot** | 支持LINE消息平台集成（v3.3版本新增统计功能） |
| **Canvas功能** | 支持Map Canvas自动识别 |

### 2.4 数字人（デジタルヒューマン）

- **自定义头像**：支持自定义Avatar的数字人形象
- **预览图像**：v3.4.0新增预览图像显示
- **X平台分享**：支持社交媒体分享功能
- **自然度优化**：持续改进数字人交互的自然程度

### 2.5 FAQ管理

- **Excel导入**：支持Excel文件批量导入FAQ
- **FAQ树结构**：支持层级化FAQ组织
- **拖拽排序**：Q&A树支持拖拽重新排列
- **自助解决**：会话式引导用户找到答案

### 2.6 企业级功能

| 功能 | 描述 |
|------|------|
| **访问权限管理** | 按部门、职位、项目精细化权限控制 |
| **本地部署** | 支持On-Premise私有化部署 |
| **丰富连接器** | 支持CRM、群件、数据库等业务系统集成 |
| **文件上传** | 最大支持500MB文件上传 |

- **缓存优化**：持续修复缓存相关问题

## 8. 竞品定位

### 8.1 产品优势
1. **日本本土化**：针对日本企业需求设计
2. **多语言自动翻译**：适合面向海外客户的日本企业
3. **数字人功能**：提供拟人化交互体验

## 9. 总结

GBase Support 是一款面向日本市场的企业级AI知识代理产品，具有以下特点：

**核心价值**：
- 基于企业自有数据构建专属AI助手
- 实现客户服务和内部知识查询的自动化
- 多语言自动翻译降低跨国服务成本

**技术亮点**：
- RAG + LLM 混合架构
- 持续的性能优化（响应速度、加载时间）
- 数字人/Avatar 拟人化交互

**商业模式**：
- SaaS订阅制，从免费试用到企业定制
- 可选实施支持服务
