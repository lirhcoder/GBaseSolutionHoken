### 2. 認定代理店一覧

### 3. GBase Support 製品調研報告
- **ファイル**: `GBaseSupport調研報告.md`
- **内容**: GBase Support製品の機能・特徴・価格体系の詳細分析

- **ファイル**: `保険業界調研報告_GBaseSupport.html`
- **内容**: 保険業界の課題とGBase Support機能のマッチング分析レポート

  - 32機能のマッチング評価
  - 41社の企業分類と推奨プラン

### 5. Insurance Painpoint Analyzer Skill
- **ファイル**: `insurance-painpoint-analyzer.zip`
- **内容**: Claude Code用のスキルパッケージ
- **機能**: 保険業界の課題調査とGBase製品マッチングレポート自動生成

## 使用方法

### HTMLレポートの閲覧
`保険業界調研報告_GBaseSupport.html` をブラウザで開いてください。
