GBase Release Note
 (Support&amp;OnPrem)
3.4.5（2025/12/19）(support)
新功能
知识库集成：新增 Box Connector 和 SharePoint Connector，支持在知识库中通过 Knowledge 授权连接 Box 和 SharePoint Project，扩展数据源。
新增断开连接功能。
文件学习队列升级到工作流引擎，提升处理能力。
Bug 修复
修复 FAQ导入数量限制的问题。
系统优化
FAQ-RAG 优化：优化 FAQ 管理机制，将其整合至统一的 RAG（检索增强生成）工作流，显著提升了问答响应的准确性与数据检索效率。
新機能
ナレッジベース連携：新しい Box Connector および SharePoint Connector を追加。Knowledge 認証を通じてナレッジベースから Box および SharePoint Project への接続をサポートし、データソースを拡張。
接続解除機能を追加。
ファイル学習キューをワークフローエンジンにアップグレードし、処理能力を向上。
バグ修正
FAQ のインポート数制限に関する問題を修正。
システム最適化
FAQ-RAG の最適化：FAQ 管理メカニズムを最適化し、統一された RAG（Retrieval-Augmented Generation、検索拡張生成）ワークフローに統合することで、質問応答の精度とデータ検索効率を大幅に向上。
